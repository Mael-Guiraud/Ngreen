Detailed Description

Représente un paquet de message circulant dans l'anneau : contient à la fois les coordonnées graphiques du messages, l'indice du noeud auquel il appartient, l'id du paquet le représentant graphiquement ainsi que la taille du paquet.

Il contient aussi un tableau contenant les messages du paquet.
